# Yeni Özellikler Güncelleme Notları (v2.0)

## 1. ElevenLabs Entegrasyonu (FR 3.3)
Sistemdeki zayıf gTTS vokalleri, **ElevenLabs** ile değiştirilmiş; gerçek zamanlı çoklu stil desteği sağlanmıştır.

## 2. Live Voice & Speech-to-Speech (FR 1.8)
Microfon ile gelen sesler Google SpeechRecognition modülü veya doğrudan ElevenLabs Voice Conversion API üzerinden hedeflere dönüştürülür.

## 3. Video Meeting Module (FR 2.9, 3.1)
İnsan ve yapay zeka aktörlerinin birleşebildiği sanal toplantı yönetimi eklenmiştir. Otomatik loglama mevcuttur.

## 4. Video Yükleme ve Çevrimdışı Swap
MP4 videolardaki TÜM yüzler işlenerek indirilebilir mp4 modeline render alınır.
